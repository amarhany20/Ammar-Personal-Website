Animals.ai Farm-Api

Programming Languages: Python
Database: Firebase, Google Storage

Libraries:
Flask, Flask-RestX, Flask-Limiter, python-dotenv, pytz, requests, google, google-cloud-logging, google-cloud-storage, gunicorn, PyJWT, jose, google-cloud-firestore, Flask-HTTPAuth, os, json, random, hashlib, datetime


The farm-api project is a robust RESTful API developed using Python Flask and deployed on Google Cloud Run. Leveraging Google Cloud Firestore and Google Storage Buckets for database storage, this API serves as a crucial backend component for storing data collected from Nvidia Jetson Devices deployed in farms. Two layers of authentication, including Token-based and Basic Authentication, ensure maximum security. Additionally, the implementation of DDOS protection with both Limiter and Google's security measures fortifies the API against potential threats. Built with a focus on scalability and efficiency, the project utilizes environment variables (.ENV) for seamless deployment across different farm locations, with automatic endpoint activation based on subscription status. A key feature of the project is its impeccable reliability and debugging capabilities, facilitated by integration with Google Logging API for real-time monitoring and error tracking. Continuous updates and deployment across all farms are automated through Docker, Google Artifact Registry, and GitHub integration. The project's core functionality revolves around receiving, sorting, validating, and storing JSONs and images, preparing them for seamless integration with front-end applications.

Animals.ai Farm-Estrus-Id-Score

Programming Languages: Python

Libraries:
Flask, Flask_Cors, numpy, opencv_contrib_python, opencv_python, pandas, python-decouple, pytz, Requests, supervision, ultralytics, google-cloud-storage, gunicorn, dill, python-dotenv, os, json, random, hashlib, datetime


The farm-estrus-id-score project, developed in collaboration with a colleague, is a RESTful API designed to process data and apply an ID Multi-Model. The ID Multi-Model, trained using Ultralytics YOLOv8, detects specific features within the data. Subsequently, a scoring function evaluates the accuracy of the detections, contributed by my colleague. Leveraging Google Logging API for real-time monitoring, the project dynamically filters confidence levels and selects the appropriate ID Model for processing. The processed data is then posted back to the Farm-API. Engineered for cost-efficiency and scalability, the project is optimized for deployment on Google Cloud Run, with data storage managed through Google Cloud Buckets. Notably, all aspects of ID Model training, data preparation, training, downloading, testing, and validation were meticulously handled by myself.

Animals.ai Farm-estrus-Jetson

Programming Languages: Python
Database: Firebase, Google Storage

Libraries:
CV2, ultralytics, supervision, numpy, PyJWT, Requests, os, json, random, hashlib, datetime, pytz, threading, FFMPEG, GStreamer


The Farm-estrus-Jetson project is a Python-based application designed to run an AI Estrus Alert system on Jetson devices. Engineered for performance optimization, the multithreaded app efficiently connects to multiple cameras via RTSP and initiates predictions for each camera. Upon detecting any alerts, the system posts them to the Farm-API. Leveraging CUDA for hardware acceleration and GStreamer for hardware-accelerated OpenCV capture, the project ensures maximum performance. Additionally, Opencv was compiled on Jetson using CMake for enhanced efficiency. The AI model, based on YOLOv8, was retrained and prepared from YOLOv2 by myself. It features dynamic confidence capabilities, allowing each camera to have its own confidence limit for better real-life deployment and improved results.

Animals.ai IoT Gate

Programming Languages: Python

Libraries:
time, PySerial, datetime, pytz, OpenCV2, threading, os


The animals.ai IOT Gate application is an IoT app designed for deployment in farms, specifically supporting EU RFID Tag Readers for automatically reading each cow's European ID. This versatile app is compatible with both RTSP and USB cameras, offering support for Linux and Windows platforms, including Jetson devices and industry computers. It seamlessly decodes real-time data from the RFID Gate and retrieves the cow's tag using PySerial. Leveraging multithreading, the app efficiently connects to multiple cameras. It serves as a data collection tool for AI and computer vision models and features an automated data storage system. Additionally, the app supports CUDA hardware acceleration on GStreamer for enhanced performance.
